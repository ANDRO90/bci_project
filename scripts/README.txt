1) To properly run all the scripts the following packages must be present in the pc:
python >= 2.7.11
numpy
matplotlib
scipy
h5py
sklearn
pylearn2

2) To properly run all the scripts it is necessary to download the datasets provided in the competition page, whose url is written in the report
3) Once the datasets have been placed in the folder containing all the scripts it is necessary to run the "spectral_estimation.py" script
4) From now on all the other scripts can be run
